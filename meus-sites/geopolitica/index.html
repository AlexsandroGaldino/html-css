O que é Geopolitica?

Se você esta aqui provavelmente você não sabe sobre Geopolítica ou deve ter pelomenos uma idea do que é, bem estamos aqui para explicar de uma maneira simples para você. Pois bem, Geopolitica é um estudo utilizado para entender os fatos atuais e o desenvolvimento dos paises. Atravez das relções estrategicas entre o poder político e os espaços geograficos das nações.

Como ela é usada?

A Geopolítica é usada em todos os países por pessoas fora e dentro dos governos, seja para prever um possivel ataque e conseguir se defender, para entender as defesas do inimigo e conseguir ataca-lo e também em questões financeiras que são as principais preocupações dos países.

Quais são os países que se beneficiam dela?

Dificilmente seria possivel dizer quantos países se beneficiam da geopolítica, mas uma coisa é certa existem países que mais se beneficiam dela são eles: Russia, Estados Unidos, China, Alemanha
